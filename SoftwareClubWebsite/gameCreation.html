Game Creation

Below are a collection of images from projects that the game creation team has worked on. If you are interested in reading about the game jam or finding a link to the final project, it is located in the events section

GMTK Game Jam

For the Game Maker's Toolkit Game Jam we created a game revolving around a robot who is seeking to break free from the control group because the theme of the event was "out of control". Our team for this game jam created all of our own art, sound effects, and code to create this 2D platformer





Beginners Circle Jam #4

The theme for the Beginners Circle Jam #4 was "game show". Since we wanted to have a quirky take on the topic, we created a game where the player controls a dear (a form of game) and shows it the path away from a hunter. Not everyone got the complex take on game show, but those who did were amazed
